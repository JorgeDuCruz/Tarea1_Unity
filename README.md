# A mayores del tutorial

Lo organice todo en un objeto vacio que tomo como tablero principal y luego añadi una copia movida con paredes, una rampa para acceder a una zona cerrada y un portal para volver a al tablero principal.

El objetivo es que cada tablero tenga un obstaculo distinto y el principal sea un "menú de seleccion".

Pero de momento falta añadir más obstaculos.
